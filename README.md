# Quiz_using_python

A console-based computer fundamentals quiz game built in Python, with score tracking and performance feedback.

## 📋 Description
This project asks the user a series of 10 questions related to computer basics (CPU, RAM, HTML, LAN, IP, etc.). The player starts with 10 marks, and 1 mark is deducted for every incorrect answer. At the end of the quiz, the final score is displayed along with a performance message.

## ✨ Features
- Interactive welcome and opt-in/opt-out flow
- 10 computer fundamentals questions
- Real-time scoring system
- Case-insensitive answer checking
- Performance feedback based on final score

## 🛠️ Tech Stack
- Python 3

## ▶️ How to Run
```bash
python Quiz_Project.py
```
